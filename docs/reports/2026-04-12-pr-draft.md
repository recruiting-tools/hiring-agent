## Summary

This PR bundles the current groundwork for HH integration and the research/evaluation artifacts needed to translate real legacy vacancies into V2 pipeline data.

- add HH mock + real HH HTTP client groundwork
- add OAuth callback and internal poll endpoint wiring
- add token refresh handling
- add legacy baseline and V2 evaluation reports built from real old DB jobs/candidates
- improve projected recruiter copy with softer CTAs
- model homework flow to completion in evaluation examples
